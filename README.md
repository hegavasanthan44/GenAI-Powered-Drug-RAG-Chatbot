# GenAI-Powered-Drug-RAG-Chatbot
Project Overview  Developed a Retrieval Augmented Generation (RAG) chatbot capable of answering drug related queries from PDF documents using Generative AI techniques.
Key Features
Implemented document ingestion and text chunking for efficient document processing
Used semantic search and vector embeddings for intelligent information retrieval
Integrated ChromaDB for vector storage and fast query retrieval
Connected Gemini API for context aware AI generated responses
Built an interactive Streamlit interface for smooth user interaction and document querying
Tech Stack
Python
Streamlit
ChromaDB
Gemini API
NLP
Vector Embeddings
Use Case

This application helps users retrieve accurate drug related information from uploaded PDF documents through an AI powered conversational interface.
